 # K V Samana – Portfolio

This is a simple, responsive personal portfolio site for **K V Samana**, a Computer Science Engineering student interested in building efficient software solutions using data and AI.

## Tech stack

- **HTML5** for structure  
- **CSS3** for layout, theming, and responsiveness (`styles.css`)  
- **JavaScript** for interactivity (`script.js`)

## Sections

- **Home**: Hero section with name, title, and quick links  
- **About**: Brief introduction and interests  
- **Education**: Academic background and scores  
- **Skills**: Core technologies and strengths  
- **Projects**: Highlighted projects focused on real-world problem solving  
- **Hackathons**: Participation and achievements  
- **Contact**: Email, phone, LinkedIn, and GitHub links

## Running locally

1. Download or clone this repository.  
2. Open `index.html` in your browser (double-click or `Open with > Browser`).  
3. Make sure `styles.css`, `script.js`, and the `assets` folder stay in the same directory as `index.html`.

## Customization

- Update content in `index.html` (text, links, projects).  
- Adjust design and colors in `styles.css`.  
- Extend behavior (e.g., animations, more interactions) in `script.js`.
