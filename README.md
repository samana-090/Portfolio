# K V Samana – Portfolio

A responsive personal portfolio built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

## Tech stack

- **React 18** for the UI
- **Vite** for fast builds and dev server
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Lucide React** for icons

## Sections

- **Home**: Hero with name, title, and CTAs
- **About**: Introduction and interests
- **Education**: Academic background
- **Skills**: Core technologies
- **Projects**: Featured work
- **Hackathons**: Participation
- **Contact**: Email, phone, LinkedIn, GitHub

## Running locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output goes to `dist/`. Deploy that folder to any static host.

## Profile image

Put your photo at `public/profile.png` (or use `profile.jpeg` and update the `src` in `src/Index.jsx`).
